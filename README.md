# PardusAI

PardusAI is an AI-powered platform that helps users to generate and edit presentation slides efficiently.

Live demo: https://pardusai.org

## Features
- AI-generated slide outlines/ document
- Editable slides directly in the browser
- Export to PDF

## Use cases
- Data analysis
- Academic presentations
- Business reports
- and more...
